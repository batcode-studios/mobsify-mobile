  ### Steps to run Locally

  ``` bash
  git clone https://github.com/batcode-studios/filesify-desktop.git

  # install dependencies
  npm install

  # serve with hot reload at localhost:9080
  npm run dev

  # build electron application for production
  npm run build
  ```
